## Installation

Download it with bower:

```sh
bower install --save pct-xxx
```


Include it in your site
```html
<!-- dependencies -->
<script src="angular.js"></script>



<!-- main include -->
<script src="pctXXX.js"></script>
```




Include it in your Angular app
```javascript
angular.module('myApp', ['pctXXX']);
```
